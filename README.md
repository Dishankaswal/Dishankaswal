### 💬 [DocChat](https://github.com/Dishankaswal/DocChat) — Live: [my-docchat.netlify.app](https://my-docchat.netlify.app)

> AI-powered RAG application — upload PDFs and chat with your 
> documents using semantic vector search and Google Gemini AI

- 📄 Upload `.pdf` or `.txt` files → text extracted and 
  split into chunks
- 🧠 Embedding Generation — Gemini API converts chunks 
  into vectors
- 🔍 Cosine similarity search retrieves the most relevant context
- 🤖 RAG-Powered Chat — Gemini generates responses using 
  retrieved document context, not just generic AI answers
- 🔐 Secure user authentication via Supabase Auth
- 💬 Chat history — view, manage, and revisit previous 
  conversations
- 📱 Fully responsive UI — optimised for desktop and mobile
- **Stack:** React · Vite · Supabase · PostgreSQL · Gemini API 
  · Tailwind CSS · Netlify
